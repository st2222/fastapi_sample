# nippo-api

## Project setup

### Install pyenv(Mac)
```
brew install pyenv
```

### Install pipenv(Mac)
```
brew install pipenv
```

### Build python environments
```
pyenv install 3.9.5
pipenv --python 3.9.5
pipenv shell
```

### Install Pipfile.lock
```
pipenv sync
```

### Run
```
pipenv run start
```

### Access
```
http://127.0.0.1:8000
```
Swagger
```
http://127.0.0.1:8000/docs
```

## Enviroment
* Python Ver 3.9.5
* pyenv Ver 2.0.1
* pipenv Ver 2021.5.29
* mysql  Ver 8.0.25
